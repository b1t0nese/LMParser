# Кинотеатры с отчётами и рекламой

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Владельцу кинотеатров, нашему старому знакомому, потребовалось расширение функционала созданной вами системы. Необходимо добавить формирование различных документов в форматах `d`ocx, `x`lsx и `p`ptx. Это будут расписания сеансов за прошедший месяц, графики загруженности кинотеатров в зависимости от времени суток, рекламные буклеты для новых кинолент с указанием кинотеатров, в которых они будут идти...

И чем больше и красочнее, тем довольнее будет заказчик.

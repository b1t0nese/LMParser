# Камень, ножницы, бумага, ром, пират

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Уже знакомые нам два пирата оказались недовольны слишком большой вероятностью получить «ничью» при жеребьёвке. И придумали для повышения результативности добавить ещё две «фигуры»: бутылку рома и пирата. По новым правилам в бумагу по-прежнему можно завернуть камень. Камень разбивает бутылку рома, от которого пьянеет пират. Пират ломает ножницы, открывающие бутылку, ром из которой размачивает бумагу о вознаграждении, назначенном за голову пирата. Пират выбрасывает камень в море, а камень, разумеется, затупляет ножницы. Теперь дело за малым: нужна новая версия программы.

## Формат ввода
Входные данные по-прежнему состоят из двух строк. На первой записан выбор первого пирата: `«`камень», `«`ножницы», `«`бумага», `«`ром» или `«`пират». На второй строке один из этих же пяти вариантов, выбранный вторым пиратом.

## Формат вывода
Вывести, как и в предыдущей версии, нужно результат жеребьёвки, то есть `«`первый», `«`второй» или `«`ничья».

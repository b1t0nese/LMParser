# В фокусе

**Ограничение времени:** 20 секунд
**Ограничение памяти:** 64Mb

Часто на картинках или фотографиях есть лишний фон, который можно без сожаления «обрезать». Начнём с простого случая, когда фон однотонный и изображение на картинке со всех сторон окружено фоном хотя бы на один пиксель. Будем вырезать из исходной картинки и сохранять минимальный прямоугольный (стороны прямоугольника параллельны краям исходной картинки) фрагмент, содержащий все пиксели, не являющиеся фоном.

## Формат ввода
Непустой файл `i`mage.png с исходной картинкой — RGB-изображением (без альфа-канала).

## Формат вывода
Файл `r`es.png с картинкой-результатом.

## Примечания
Обратите внимание на метод [`c`rop](https://pillow.readthedocs.io/en/stable/reference/Image.html#PIL.Image.Image.crop), который умеет создавать новую картинку, вырезая прямоугольник из заданной.

По этой [ссылке](http://lyceum.s3.yandex.net/content/resources/2.1/v%20fokuse.zip) можно скачать все тестовые пары для этой задачи.

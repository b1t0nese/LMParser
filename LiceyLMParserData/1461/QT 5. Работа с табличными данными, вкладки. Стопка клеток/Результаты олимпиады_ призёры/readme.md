# Результаты олимпиады: призёры

**Ограничение времени:** 10 секунд
**Ограничение памяти:** 64 Мб

Модифицируйте задачу *«Результаты олимпиады: фильтрация»* таким образом, чтобы после применения фильтрации выделялись цветами участники, которые заняли при выбранном фильтре 1, 2 и 3 место.

Если несколько участников набрали одинаковое количество баллов, то считается, что они заняли одно и то же место, таким образом может быть несколько первых мест, несколько вторых и несколько третьих. Выводить необходимо только логин, ФИО и суммарный балл.

Класс, реализующий окно приложения, наследуется от `QWidget` и называется *OlympResultWin*. Виджет `QTableWidget`, содержащий таблицу, назовите *tableWidget*.

Выпадающие списки нужно реализовать с помощью `QComboBox` и назвать_ classes_ и *schools* для соответствующего выбора.

Кнопку для фильтрации назовите *resultButton*. Цвет для первого места #CCCC00, второго #B5B5BD, третьего #9C5221.

Файл с данными **rez.csv** можно скачать по [ссылке](https://yastatic.net/s3/lyceum/content/problems/qt6/rez.csv).

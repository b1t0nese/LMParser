# Осторожно: дорого

**Ограничение времени:** 100 секунд
**Ограничение памяти:** 64 Мб

В файле [price.csv](https://assets.contest.yandex.net/testsys/statement-file?hash=eyJhbGciOiJkaXIiLCJlbmMiOiJBMjU2R0NNIn0..HbmQ9V0dPhcIA1YN.1skDEDDV35wfyP97cpbq3VMzBgr5YU6FeeueVrhFnQSprfgW-l_i9ZHQsbxTkDmlVyx2VVWN7QmuADzjUE01QgjW7ByU_h_cLzds.koii7lR3H3y9xmR-NSXjNQ) содержатся цены на товары некоторого магазина через разделитель — точку с запятой.

Напишите приложение на PyQT6, отображающее содержимое этого файла в виде таблицы, отсортированной по стоимости товара (от более дорогого — к дешевым).
Таблицу поместите в виджет `QTableWidget` с именем *tableWidget*.

Товары, имеющие цену, входящую в ТОП-5 самых высоких, выделите разными случайными цветами.
По нажатию на кнопку с именем *updateButton* должно происходить обновление цветов.
Класс, реализующий окно приложения и наследующийся от `QWidget`, назовите **Expensive**.

# Точка обыкновенная

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Реализуйте класс `P`oint — точка на плоскости. При создании точка получает два параметра: целочисленные координаты по оси абсцисс и оси ординат. Кроме того, точка умеет возвращать свои координаты по отдельности с помощью методов `g`et_x и `g`et_y соответственно и двухэлементным кортежем при использовании метода `g`et_coords.

# Именованная точка

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Добавим точкам класса Point имена. При инициализации имя будет указываться в качестве первого параметра Point(name, x, y).

Определим с учётом этого нововведения метод получения строкового представления точки таким образом, чтобы возвращаемая строка состояла из имени точки и её координат в скобках с запятой и пробелом между ними.

А также реализуем операцию инверсии «l. 51 ∼». Её применение должно будет создавать точку, имя которой совпадает с именем исходной точки, а координаты переставлены местами, т.е. инверсией точки l. 53A(x,y) будет точка l. 53A(y,x).

## Формат ввода
Каждый тест представляет собой код, в котором будет использоваться ваш класс.

## Примеры
### Пример 1
**Ввод:**
```
from solution import Point

point_A = Point('A', 3, -4)
print(point_A)
point_B = ~point_A
print(point_B)
print(~Point('O', 0, 0))
```
**Вывод:**
```
A(3, -4)
A(-4, 3)
O(0, 0)
```

## Примечания
Операция инверсии задаётся с помощью метода __invert__.

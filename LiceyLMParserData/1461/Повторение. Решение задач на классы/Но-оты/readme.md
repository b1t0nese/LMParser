# Но-оты

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Продолжаем проектировать класс Note. Кроме высоты, у звука, представленного нотой, есть ещё и длительность. В нашем случае вариантов будет всего два: обычный (короткий) звук или длинный. При создании ноты длительность задаётся необязательным вторым параметром. True — нота длинная, False или параметр не задан — короткая. Длинная нота отличается от короткой тем, что при воспроизведении она звучит как «до-о», «ре-э», «ми-и», «фа-а», «со-оль», «ля-а» или «си-и» соответственно. Кроме того, для воспроизведения мы больше не будем использовать специальный метод, а реализуем метод __str__, возвращающий строковое представление объекта.

## Формат ввода
Каждый тест представляет собой код, в котором будет использоваться ваш класс.

## Примеры
### Пример 1
**Ввод:**
```
from solution import Note

do_1 = Note("до", False)
doo = Note("до", True)
do_2 = Note("до")
print(do_1, doo, do_2)
```
**Вывод:**
```
до до-о до
```

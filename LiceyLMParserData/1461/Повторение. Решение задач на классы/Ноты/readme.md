# Ноты

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Реализуйте класс `N`ote, моделирующий ноту детского ксилофона на одну октаву. При создании нота получает одну из семи возможных высот: `«`до», `«`ре», `«`ми», `«`фа», `«`соль», `«`ля» или `«`си». Кроме того, нота имеет метод `p`lay, при вызове которого печатает свою высоту.

## Формат ввода
Каждый тест представляет собой код, в котором будет использоваться ваш класс.

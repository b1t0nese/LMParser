# Танцы шариков

**Ограничение времени:** 20 секунд
**Ограничение памяти:** 512 Мб

Напишите приложение для `arcade`, в котором 12 белых мячиков заданного радиуса возникают в центре окна и разлетаются в разные стороны, которые определяются различиями в их скоростях: 4 шарика имеют всевозможные сочетания скоростей (1, -1) по x и по y, и ещё 8 — сочетания (-2, -1, 1, 2).

Мячи летят до границ окна, от которых отражаются и меняют скорость по оси на противоположную.

Цвет окна DARK_BLUE_GRAY. Для рисования шариков используйте метод `draw_circle_filled`.

Используйте шаблон кода ниже (вместо `...` впишите свой код). Атрибуты класса должны иметь такие же имена, как и переданные или объявленные аргументы в шаблоне.

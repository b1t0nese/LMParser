# Высота над горизонтом


Из-за того, что Земля круглая, на разных широтах Солнце поднимается над горизонтом на разную высоту. В зените оно может оказаться в полдень только на экваторе, на других широтах оно будет ниже.

Чтобы определить высоту Солнца над горизонтом (а по высоте уже можно узнать широту местности), используют гномон — прибор в виде вертикального шеста. Пользоваться им просто: надо определить длину кратчайшей тени, а затем, зная высоту шеста, вычислить противолежащий ему угол прямоугольного треугольника.

## Формат ввода
Вводятся два числа через пробел: высота гномона и длина кратчайшей тени.

## Формат вывода
Выведите угол — высоту Солнца над горизонтом, округлённый до целых.

Обработайте возможные исключения:

- невозможно прочитать два введённых значения через пробел; выводится сообщение:
`Wrong input data: <стандартное сообщение об ошибке>`
- невозможно преобразовать ввод в целые числа; выводится сообщение:
`Wrong input data: <стандартное сообщение об ошибке>`
- высота Солнца отрицательная; выводится сообщение:
`Negative sun height.`

## Примеры
### Пример 1
**Ввод:**
```
2025
```
**Вывод:**
```
Wrong input data: not enough values to unpack (expected 2, got 1)
```

# Маловато будет

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Ошибка `EOFError` (`EndOfFileError`) возникает тогда, когда вы пытаетесь считать ввод, а во входном файле уже ничего нет. Напишите программу, которая не упадёт, если вводимых строк недостаточно.

## Формат ввода
Вводятся 6 строк (но это не точно).

## Формат вывода
Выведите слово, которое получится, если из каждой введённой строки взять среднюю букву (или ту, что левее середины, если длина строки чётная).

Но если введено не шесть строк, а меньше, перехватите ошибку и выведите сообщение:
 `Error EOF, <количество строк> of lines entered.`

Не используйте `if`, используйте конструкцию `try ... except`.

## Примеры
### Пример 1
**Ввод:**
```
were
penny
not
found
begin
the
```
**Вывод:**
```
enough
```

# Буква из алфавита

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Напишите программу, которая находит букву в алфавите по её номеру.

Использовать `try ... except` обязательно.

## Формат ввода
Вводится строка — `RUS` или `ENG` — из какого алфавита выбирается буква. Затем число — номер буквы в алфавите.

## Формат вывода
Если букву получить невозможно, ваша программа не должна падать с ошибкой, а выводить сообщение:
 `Буквы под номером <номер> в алфавите <алфавит> нет.`

Если исключения не было, выводится сообщение:
 `Буква <буква> стоит в алфавите <алфавит> под номером <номер>.`

## Примеры
### Пример 1
**Ввод:**
```
RUS
7
```
**Вывод:**
```
Буква Ё стоит в алфавите RUS под номером 7.
```

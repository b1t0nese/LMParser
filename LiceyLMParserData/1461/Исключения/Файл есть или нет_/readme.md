# Файл есть или нет?

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В файле с заданным именем скрывается список словарей. Если из каждого словаря взять значение по указанному ключу, то должно получиться важное сообщение. Но без подвоха не обойтись.

Может не оказаться файла в директории.
 В словаре может не оказаться указанного ключа.
 Значением по ключу может быть не строка.

Напишите программу, которая не упадёт, даже если случится одно из этих событий.

В первом случае выведите сообщение: `There is no such file in the directory.`
 Во втором — `The key is missing.`
 В третьем — `It is impossible to add non-line with line.`

И только если ничего неожиданного не произошло, составьте предложение из значений словарей через пробел в исходном порядке.

## Формат ввода
Вводится имя файла и название ключа.

## Формат вывода
Выведите сообщение. Не используйте `if`, используйте конструкцию `try ... except`.

В примере используется файл **wisdom.json** с таким содержимым:

```
[
 {
   "think": "To be",
   "word": "Now"
 },
 {
   "guide": "Wisdom",
   "think": "or",
   "word": "or"
 },
 {
   "word": "never",
   "think": "not to be"
 }
]
```

## Примеры
### Пример 1
**Ввод:**
```
wisdom.json
word
```
**Вывод:**
```
Now or never
```

# Посторонним В

**Ограничение времени:** 2 секунды
**Ограничение памяти:** 64 Мб

Вход воспрещён тем, у кого оказалось ровно пять простых делителей.

Напишите функцию **strangers()**, которая принимает произвольное число целых чисел, и возвращает список тех из них, которым вход разрешён (в порядке неубывания).

Если аргументы не переданы, возбуждается собственное исключение `NoArgsError` с сообщением: *The arguments are not passed.*
 Если среди аргументов попались не числа, возбуждается системное исключение `ValueError` с сообщением: *There are not only numbers among the arguments!*
 Если возвращаемый список пуст, возбуждается собственное исключение `EmptyListError` с сообщением: *There were no suitable numbers.*

Порядок проверки исключений должен соответствовать описанию в задаче.
 **Исключения нужно только порождать, перехватывать их будет тестирующая система!**

## Примеры
### Пример 1
**Ввод:**
```
data = [95677675, 6021226791, 1988275043,
        199535748661679, 6188627638849]
print(strangers(*data))
```
**Вывод:**
```
[95677675, 6188627638849]
```

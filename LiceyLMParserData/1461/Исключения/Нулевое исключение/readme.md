# Нулевое исключение

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Необходимо разделить число на произведение его цифр. Напишите программу так, чтобы она не падала при попытке деления на ноль.

Использовать `if` нельзя, а вот `try ... except` — обязательно.

## Формат ввода
Вводится число.

## Формат вывода
Выведите отношение числа к произведению его цифр, если его можно найти, а если нельзя, то 0.

## Примеры
### Пример 1
**Ввод:**
```
12345678
```
**Вывод:**
```
306.1924107142857
```

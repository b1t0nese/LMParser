# Тестирование взятия ферзём

**Ограничение времени:** 10 секунд
**Ограничение памяти:** 64Mb

С помощью библиотеки `pytest` напишите тесты для функции `is_under_queen_attack(position, queen_position)`,
      которая возвращает `True`, если шахматное поле с координатой `position` находится под ударом ферзя,
      находящегося на поле с координатой `queen_position`, и `False` в противном случае. Считается, что ферзь
      бьет поле, на котором стоит.

Координаты задаются строками в стандартной шахматной нотации для доски 8x8: строчная буква и цифра, например `"e2"`.

Если хотя бы одна из координат не является строкой, должно выбрасываться исключение `TypeError`.

Если хотя бы одна из координат не является правильной координатой для доски 8x8 (например, `"j5"` или `"abc"`),
      должно выбрасываться исключение `ValueError`.

Корректность значений необходимо проверять **последовательно**: сначала все условия для первой координаты,
      а затем — все для второй.

Функцию `is_under_queen_attack` нужно импортировать из модуля `yandex_testing_lesson`.

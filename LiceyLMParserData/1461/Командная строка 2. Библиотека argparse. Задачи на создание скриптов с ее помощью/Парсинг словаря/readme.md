# Парсинг словаря

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Иногда в качестве параметров в командной строке передаются данные вида "ключ=значение", причем количество таких параметров
      заранее неизвестно.
 Напишите программу, которая разбирает такие параметры и выводит их на экран в виде:
 "Key: ключ
      Value: значение",

 при этом каждая пара выводится на отдельной строке.
 Еще программа должна принимать опцию --sort,
      которая позволяет упорядочивать выводимые значения по ключу.

 И конечно же, вам необходимо использовать библиотеку


      a


      r


      g


      p


      a


      r


      s


      e


      argparse


argparse.

## Примеры
### Пример 1
**Ввод:**
```
python3 solution.py --sort name=Vasya surname=Ivanov age=25
```
**Вывод:**
```
Key: age	Value: 25
Key: name	Value: Vasya
Key: surname	Value: Ivanov
```

## Примечания
Проверка валидности входных параметров не требуется.

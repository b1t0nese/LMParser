# Предсказатель пола

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Представьте, что вам заказали написать программу, которая по ответам пользователя умела бы определять его пол. Необходимо
      написать скрипт, который в качестве аргументов принимает несколько факторов (именованных аргументов):

- "barbie" — говорит об отношении пользователя к куклам. Это целое число в диапазоне от 0 до 100, значение по умолчанию
      — 50;
- "cars" — говорит об отношении пользователя к машинам. Значение задается аналогично предыдущему.
- "movie" — это любимая телевизионная программа пользователя из списка **[melodrama, football, other]**.
      По умолчанию — **other**.

На выходе же скрипт должен выводить вероятность мужского и женского пола пользователя от 0 до 100% по формуле

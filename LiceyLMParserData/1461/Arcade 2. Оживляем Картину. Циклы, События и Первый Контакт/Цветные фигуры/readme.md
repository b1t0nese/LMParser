# Цветные фигуры

**Ограничение времени:** 20 секунд
**Ограничение памяти:** 1024 Мб

Реализуйте приложение на `arcade`, в котором по клику левой кнопкой мыши рисуется **круг**, а по клику правой — **квадрат**. Диаметр круга и сторона квадрата должны быть равны **20** пикселям.

Цвета фигур циклически чередуются из списка, передаваемого в параметрах. При удерживании клавиши `Ctrl` во время клика фигура рисуется **белой**, однако счётчик для чередования основных цветов всё равно должен продвигаться. Точка клика - геометрический центр круга или квадрата.

# Ямы на дороге

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Подготовьте задание ремонтной бригаде: где какие выбоины в асфальтовом покрытии образовались, какие их параметры, какой приоритет ремонта.

## Формат ввода
Вводится контрольное число: если произведение площади поверхности повреждённого покрытия на глубину больше этого числа, то приоритет ремонта 1, иначе 0.
Затем вводится количество ям и для каждой ямы информация в формате:
`<километр> <полоса> <площадь> <глубина>`

## Формат вывода
В файл `holes.csv` запишите техническое задание на ремонт с заголовками (разделитель — точка с запятой):
`номер по порядку, километр, полоса, площадь, глубина, приоритет`
`no, kilometer, stripe, area, depth, priority`

## Примеры

**Ввод:**

```
7

```
**Вывод:**
```
no;kilometer;stripe;area;depth;priority
1;30;1;91;3;1
2;33;4;81;5;1
3;50;3;25;4;0
4;50;2;20;3;0
5;45;2;85;4;1
6;50;4;85;3;1
7;30;2;91;4;1
```

# Номер строки

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В файле `numbers.txt` записано много строк, в каждой из которых записано шесть чисел через пробел. Определите номер строки с наименьшей суммой чисел в строке, для которой выполняются условия:

- в строке только одно число повторяется трижды, остальные три числа различны;
- среднее арифметическое неповторяющихся чисел строки не меньше повторяющегося числа;
- максимальное число строки не кратно минимальному.

Если таких строк несколько, то выведите меньший номер. Нумерация строк идёт с 1.

```

```
**Вывод:**
```
18
```

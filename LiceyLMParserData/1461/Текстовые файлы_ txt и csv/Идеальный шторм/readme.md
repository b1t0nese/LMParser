# Идеальный шторм

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В узком заливе во время шторма помещается ровно шесть горбов волны, но высота каждого горба волны может немного отличаться. Определите количество волн в заливе, для горбов которых выполняются условия:

- в волне ровно два одинаковых горба, остальные разные;
- в волне есть хотя бы один горб, оканчивающийся на такую же цифру, что и максимальное значение из всех горбов всех волн;
- хотя бы три горба этой волны — чётные.

## Формат ввода
В файле `storm.csv` записаны строки из шести целых чисел с разделителями — запятыми.

## Формат вывода
Выведите количество подходящих волн (строк в файле) и наименьшую высоту горба волны из таких строк через пробел.

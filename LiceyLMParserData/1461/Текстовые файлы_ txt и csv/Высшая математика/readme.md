# Высшая математика

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Напишите программу, которая автоматизирует научные расчёты, то есть для введённой функции составит и запишет в файл таблицу значений.

## Формат ввода
Вводится функция, зависящая от переменной `x`. В функции могут содержаться математические функции из библиотеки `math`, поэтому не забудьте импортировать библиотеку в виде
`from math import *`

## Формат вывода
В файл `function.txt` в двух столбцах запишите значения `x` из диапазона от 0 до 2 с шагом 0.01 и значения функции для этого `x`, округлённые до тысячных, через символ табуляции.

В примере показана часть файла ответа.

Пример ввода

```
sqrt(x)
```

Результат

```
0.0 0.0
0.01    0.1
0.02    0.141
0.03    0.173
0.04    0.2
0.05    0.224
0.06    0.245
0.07    0.265
0.08    0.283
0.09    0.3
```

## Примечания
Чтобы выполнить строку как код `Python`, передайте её функции `eval()`. Используемые в строке имена переменных должны быть определены в коде выше.

**Важно!**
Функцию `eval()` можно использовать только в учебных целях!

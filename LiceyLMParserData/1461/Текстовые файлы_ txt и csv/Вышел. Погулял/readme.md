# Вышел. Погулял

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Из записанного стишка сделайте считалочку: перед каждой строкой выводите счёт из чисел с 1 через дефис, окружённый пробелами.

## Формат ввода
Стишок из нескольких строк записан в файле `poem.txt`.

## Формат вывода
Считалочку запишите в файл `counting.txt`.

## Примеры
### Пример 1
**Ввод:**
```
Раз
Два
Три
Четыре
Пять
Вышел
зайчик
погулять
```
**Вывод:**
```
1 - Раз
2 - Два
3 - Три
4 - Четыре
5 - Пять
6 - Вышел
7 - зайчик
8 - погулять
```

# Пляж для купания

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Для открытия пляжа для купания нужно, чтобы он удовлетворял всем требованиям из перечня:

- он должен быть оборудован местами для отдыха (0 — не оборудован, 1 — оборудован),
- на нём должна работать спасательная служба (0 — нет, 1 — есть),
- загрязнение воды не должно превышать 0,5,
- температура воды не должна быть ниже 18 градусов.

## Формат ввода
В файле `beaches.csv` записана информация о состоянии нескольких пляжей с заголовками (разделитель — точка с запятой):
`id, название, оборудован, спасатели, загрязнения, температура`
`id, beach, equipped, rescuers, pollution, temperature`

## Формат вывода
Выведите названия пляжей, которые могут быть открыты для купания, в порядке перечисления в файле.

## Примеры
### Пример 1
**Ввод:**
```
id;beach;equipped;rescuers;pollution;temperature
456;Большой Садовый пруд;1;1;0.08;20
832;Золотые пески;1;1;0.5;22
172;Серебряный бор-3;1;0;0.4;19
794;Тропарёво;1;1;1.36;22
773;Головинские пруды;1;1;0.46;19
869;Динамо;0;1;1.36;27
```
**Вывод:**
```
Большой Садовый пруд
Золотые пески
Головинские пруды
```

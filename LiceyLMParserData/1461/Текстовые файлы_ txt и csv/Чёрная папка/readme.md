# Чёрная папка

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В чёрной-чёрной папке есть другая чёрная-чёрная папка, в которой есть чёрный-пречёрный файл, в котором нужно найти всех котиков, необязательно чёрных.

В папке `black` есть файлы и папки, но нас интересует только папка `another_black`, в которой есть файл `cats_black_and_other.txt`.

Прочитайте содержимое этого файла; выберите строки, в которых есть подстрока `cat` в любом регистре.

В файл `cats.txt` выведите все слова из таких строк, которые начинаются с прописной буквы, а остальные — строчные. Слова выводите в алфавитном порядке по одному.

Структура папки из примера:

Пример исходного файла

```
The mouse took credit for not eating cats.
A cat with gloves won't catch a mouse.
It doesn't matter if it's a Black cat or a White cat.
If it can catch mice, it's a good Cat.
```

Пример ответа

```
A
Black
Cat.
If
It
The
White
```

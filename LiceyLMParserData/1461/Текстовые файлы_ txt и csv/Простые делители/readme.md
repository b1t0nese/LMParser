# Простые делители

**Ограничение времени:** 3 секунды
**Ограничение памяти:** 64 Мб

Напишите программу, которая для всех чисел из заданного диапазона найдёт первые пять простых делителей (1 — не простое число; само число, если оно простое, считается делителем) и запишет их в файл `divisors.csv` с заголовками (разделитель — пробел):
 `число, делитель 1, делитель 2...`
 `number, div1, div2, div3, div4, div5`

## Формат ввода
Вводятся два числа — границы диапазона, включая концы.

## Формат вывода
В файл `divisors.csv` запишите простые делители всех чисел диапазона, включая концы. Если у числа нет пяти простых делителей, то вместо отсутствующих ничего не пишите.

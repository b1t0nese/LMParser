# Бабочки

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Бабочки повсюду! Выберите из них только тех, которые встретились ровно в двух местах, ни больше, ни меньше!

## Формат ввода
В одну строку через точку с запятой и пробел вводятся имена файлов с бабочками, записанными по одной в строке.

## Формат вывода
Выведите в произвольном порядке бабочек, которые в любом количестве встречались только в двух файлах. Выводите также по одной в строке.

## Примеры
### Пример 1
**Ввод:**
```
butt1.txt; butt2.txt; butt3.txt; butt4.txt
```
**Вывод:**
```
Small Pearl-bordered Fritillary
Glanville Fritillary
Green-veined White
High Brown Fritillary
```

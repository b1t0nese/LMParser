# Запрос 5

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Для базы данных, созданной по рассмотренным ранее моделям **User** и **Jobs**, нужно написать запрос.
 Напишите программу, которая считывает из консоли имя базы данных, подключается к ней с использованием возможностей библиотеки *sqlalchemy* и выводит работы, выполнение которых требует меньше 20 часов и которые еще не закончены, каждую с новой строки.

Для отображения результатов в модели Jobs нужно переопределить метод _repr_ по следующему примеру: <Job> название работы

## Примеры
### Пример 1
**Ввод:**
```
mars_explorer.sqlite
```
**Вывод:**
```
<Job> deployment of residential modules 1 and 2
<Job> exploration of mineral resources
<Job> analysis of atmospheric air samples
```

## Примечания
Вашей программе доступны функции *global_init(db_name)*, *create_session()*, а также классы моделей **User** и **Jobs**.

Обратите внимание, что в базе данных логический тип записывается как **0** или **1**.

# Красивый калькулятор

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64.0 Мб

Вам дан файл [calc.ui](https://yastatic.net/s3/lyceum/files/3e335edf-1a9e-49d9-b485-dfbb44eddf7e/upload.ui), в котором реализован интерфейс красивого калькулятора. Изучите его, подключите к своей программе любым из способов и реализуйте все его функции. (Загружать этот файл вместе с решением не надо, изменять интерфейс тоже)

Класс, реализующий окно приложения, назовите **Calculator**

Если какое-то выражение вычислить не получается (например, квадратный корень для отрицательного числа), должно выводиться соответствующее сообщение.

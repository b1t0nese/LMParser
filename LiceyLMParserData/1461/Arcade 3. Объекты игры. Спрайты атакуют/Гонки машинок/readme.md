# Гонки машинок

**Ограничение времени:** 30 секунд
**Ограничение памяти:** 1024 Мб

Организуйте гонку машинок в окне `arcade` размером **1200х400** с белым фоном.

У левого края окна разместите 4 машинки (файлы `car1.png` - `car4.png` из архива [cars.zip](https://yastatic.net/s3/lyceum/content/images/d2_arcade3/D_%D0%93%D0%BE%D0%BD%D0%BA%D0%B8%20%D0%BC%D0%B0%D1%88%D0%B8%D0%BD%D0%BE%D0%BA/cars.zip)), программно уменьшив их в два раза. Их правый край должен находиться на отметке **300** пикселей от левого края окна. По нажатию на `Пробел` машинки начинают гонку, двигаясь вправо со случайными скоростями. Доехав до правого края, машинка останавливается, и на финише появляется цветной прямоугольник размером **10x100**, соответствующий цвету гонщика.

# Мыльные пузыри

**Ограничение времени:** 20 секунд
**Ограничение памяти:** 896 Мб

Напишите приложение, в котором на чёрном фоне окна **800х600** в случайных местах появляются **10** пузырей.

По клику мышкой пузырь лопается: вместо него проигрывается анимация взрыва. Картинки для анимации находятся в архиве [balls.zip](https://yastatic.net/s3/lyceum/content/images/d2_arcade3/I_%D0%9C%D1%8B%D0%BB%D1%8C%D0%BD%D1%8B%D0%B5%20%D0%BF%D1%83%D0%B7%D1%8B%D1%80%D0%B8/balls.zip). Для решения вам потребуется создать класс `Bubble`, унаследованный от `arcade.Sprite`.

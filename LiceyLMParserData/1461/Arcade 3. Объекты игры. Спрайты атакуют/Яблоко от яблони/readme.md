# Яблоко от яблони

**Ограничение времени:** 30 секунд
**Ограничение памяти:** 2048 Мб

Напишите приложение, в котором на фоне с яблоней в случайных местах её кроны размещаются **10** яблок.

Крона яблони представляет собой прямоугольную область **700x500** пикселей, расположенную по центру окна **1000x800**. По клику мыши на яблоке оно начинает падать со скоростью **50** пикселей в секунду. Достигнув нижнего края окна, яблоко останавливается. Файлы для фона [tree.png](https://yastatic.net/s3/lyceum/content/images/d2_arcade3/B_%D0%AF%D0%B1%D0%BB%D0%BE%D0%BA%D0%BE%20%D0%BE%D1%82%20%D1%8F%D0%B1%D0%BB%D0%BE%D0%BD%D0%B8/tree.png) и яблока [apple.png](https://yastatic.net/s3/lyceum/content/images/d2_arcade3/B_%D0%AF%D0%B1%D0%BB%D0%BE%D0%BA%D0%BE%20%D0%BE%D1%82%20%D1%8F%D0%B1%D0%BB%D0%BE%D0%BD%D0%B8/apple.png) находятся в папке `images`.

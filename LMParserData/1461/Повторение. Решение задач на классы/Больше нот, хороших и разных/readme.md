# Больше нот, хороших и разных

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Добавим несколько разновидностей нот, унаследовав их от базового класса `N`ote.

Пусть класс `L`oudNote моделирует громкие ноты, «звучащие» капслоком (в остальном они идентичны обычным нотам класса `N`ote).

Создадим также класс `D`efaultNote, конструктор которого не считает высоту ноты обязательным параметром. Если высота не передана, по умолчанию создаётся нота «до». Чтобы у ноты можно было указать длительность, не задавая при этом высоту, нам понадобится договориться о едином названии параметра, отвечающего за длительность, пусть он называется `i`s_long во всех классах.

И ещё один класс — `N`oteWithOctave, расширяющий объекты-ноты дополнительным атрибутом — октавой, к которой относится нота. Октаву нужно принимать как обязательный аргумент при инициализации, а также при приведении к строке выводить в круглых скобках после самой ноты.

В решении нужно объявить и можно использовать список нот (тестирующая система будет импортировать его из вашего кода):

```
PITCHES = ["до", "ре", "ми", "фа", "соль", "ля", "си"]
```

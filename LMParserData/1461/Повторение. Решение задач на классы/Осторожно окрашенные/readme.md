# Осторожно окрашенные

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Добавим цвета, унаследовав от класса `P`oint новый класс `C`oloredPoint. Точки нового класса будут очень похожи на своих предшественниц. Отличия следующие:

- При инициализации дополнительно может быть передан необязательный параметр — RGB-значение цвета. Цвет задаётся трёхэлементным кортежем целых чисел от 0 до 255. По умолчанию точки чёрные: цвет равен (0, 0, 0).
- Добавляется метод `g`et_color, возвращающий значение цвета точки в том же формате, в каком он задаётся.
- При инверсии ( ∼ \sim ∼) цвет точки также инвертируется. Для этого значение каждой RGB-компоненты отнимается от 255.

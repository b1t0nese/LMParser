# Еще один запрос

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Заполните таблицу департаментов данными.

Теперь нам нужно создать запрос, в котором используются данные всех трех моделей. Вывести фамилии и имена тех работников департамента **id = 1** (геологической разведки) (или другого, который есть в вашей таблице), которые имеют суммарное количество отработанных часов *больше 25*.

## Примеры
### Пример 1
**Ввод:**
```
mars_explorer_2.sqlite
```
**Вывод:**
```
Watny Mark
Kapoor Venkat
Sanders Teddy
```

## Примечания
Вашей программе доступны функции *global_init(db_name)*, *create_session()*, а также классы моделей **User**, **Jobs** и **Department**.

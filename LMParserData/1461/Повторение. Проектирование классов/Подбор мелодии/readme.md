# Подбор мелодии

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Теперь составим из нот мелодию.

 Создайте класс `M`elody, который будет моделировать процесс
      подбора мелодии. Для этого нужно уметь:

- преобразовывать мелодию в строку, записывая ноты через запятую, первая нота с заглавной буквы,
- добавлять ноту в конец уже имеющейся мелодии — метод `a`ppend,
- заменять последнюю ноту другой — метод `r`eplace_last (без проверки её существования),
- удалять последнюю ноту — `r`emove_last (также без проверки),
- начинать всё с начала (стирать все ноты) — метод `c`lear,
- возвращать длину мелодии (количество нот в ней) при вызове функции `l`en от неё,
- а также применять упрощённое транспонирование всей мелодии вверх или вниз (сдвиги

      >


      >


      >>


>> и

      <


      <


      <<


<<), если это удаётся сделать внутри одной октавы (без циклического
      сдвига), если же хотя бы одна нота выходит за пределы октавы, возвращать копию исходной мелодии (т.к. возможности нашего ксилофона
      ограничены одной октавой и только «белыми» клавишами).

Используйте те же константы, что и для предыдущей задачи (тестирующая система будет импортировать их из вашего кода):

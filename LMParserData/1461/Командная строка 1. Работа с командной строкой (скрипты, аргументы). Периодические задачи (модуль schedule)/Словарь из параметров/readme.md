# Словарь из параметров

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Используя  s y s . a r g v sys.argv sys.argv напишите программу, которая разбирает аргументы командной строки вида "ключ=значение" и выводит их на отдельной строке в виде "Key: ключ Value: значение".
Кроме описанных аргументов программа может принимать необязательную опцию --sort, которая позволяет отсортировать выводимые значения по ключу.

## Примеры
### Пример 1
**Ввод:**
```
python3 solution.py --sort name=Vasya age=42
```
**Вывод:**
```
Key: age Value: 42
Key: name Value: Vasya
```

## Примечания
Ваша программа может иметь произвольное имя.
Проверка валидности входных параметров не требуется.
Необходимо обязательно использовать  s y s . a r g v sys.argv sys.argv.

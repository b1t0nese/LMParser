# Проверка ошибок

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64Mb

Давайте создадим простой модуль.
 В нем необходимо реализовать функцию **print_error(message)**, которая
      выводит передаваемый параметр, оборачивая его в следующий шаблон:

 ERROR: message!!

 Но самое интересное,
      что ваша программа должна получать из командной строки параметр — строку, вывести приветственное сообщение **Welcome
      to my program** и уже потом вызвать функцию **print_error()** c переданным параметром.

 Тестирующая
      система будет как просто выполнять вашу программу, так и подключать ее как модуль с помощью директивы **import**,
      а затем вызывать функцию **print_error()**.

 Поведение должно отличаться.

## Примеры
### Пример 1
**Ввод:**
```
python3 solution.py message
```
**Вывод:**
```
Welcome to my program
ERROR: message!!
```

## Примечания
Проверка валидности/наличия аргумента не требуется.
 В задаче надо обязательно использовать модуль **argparse**

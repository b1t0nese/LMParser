# Число из числа

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В процессе преобразования исходного числа нужно проверять, что получаются корректные промежуточные результаты.
 Напишите программу, которая для этих проверок использует не `if`, а `assert`.

Проверки такие:

- Введённое число должно иметь более четырёх разрядов; если это не так, то возбуждается `AssertionError` с сообщением: `There are not enough discharges.`
- Последние четыре разряда числа переставляются влево (в начало числа), при этом количество разрядов не должно уменьшиться; если это не так, то возбуждается `AssertionError` с сообщением: `The number has decreased.`
- К преобразованному числу прибавляется сумма его цифр, при этом результат должен оказаться чётным; если это не так, то возбуждается `AssertionError` с сообщением: `Odd result.`

Если все проверки пройдены, выведите результат, иначе — сообщение исключения `AssertionError`.

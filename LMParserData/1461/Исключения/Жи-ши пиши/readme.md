# Жи-ши пиши

**Ограничение времени:** 2 секунды
**Ограничение памяти:** 64 Мб

После ж, ч, ш, щ не пишутся ю, я, ы, а пишутся у, а, и, например: чудо, щука, час, роща, жир, шить. Исключения — в иноязычных словах после ж и ш могут стоять ю или я, например жюри, брошюра.

Напишите функцию **check()**, которая проанализирует переданный текст и возбудит исключение, если найдёт в нём запрещённые буквы после шипящих.

Если есть нарушение правила, то нужно возбудить собственное исключение `BrokenRuleError` с сообщением: *The rule is broken.*

Если все слова с шипящими записаны верно, то нужно вернуть сообщение: *That's right.*

**Исключения нужно только порождать, перехватывать их будет тестирующая система!**

# Системный перевод

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Напишите программу, которая переводит числа, записанные в произвольной системе счисления от 2 до 36, в десятичную систему.

Использовать `if` нельзя, `try ... except` обязательно.

## Формат ввода
Вводятся строки: число и через пробел система счисления, в которой оно, вероятно, записано.

## Формат вывода
Для каждой строки выводите ответ в виде:
 `<исходное число>(<основание>) = <перевод в десятичную систему>`

Если перевести число невозможно, пропускайте такую строку.

## Примеры
### Пример 1
**Ввод:**
```
1001110 2
123 3
5602 8
9A30F 26
```
**Вывод:**
```
1001110(2) = 78
5602(8) = 2946
9A30F(26) = 4290587
```

# Не чирикайте

**Ограничение времени:** 2 секунды
**Ограничение памяти:** 64 Мб

Напишите функцию, выбирающую все звуки, кроме запрещённых.

Функция **chirp()** принимает произвольное число строк и именованный аргумент `taboo` с запрещённой подстрокой, который может быть и не передан. Возвращается список строк, в которых нет запрещённой подстроки ни в каком регистре, порядок строк — алфавитный.

Если именованный аргумент не передан, возбуждается собственное исключение `NoSubstringError` с сообщением: *No named argument.*
 Если хотя бы один позиционный аргумент не является строкой, нужно возбудить исключение `TypeError` с сообщением: *The argument is not a string.*

Порядок проверки исключений должен соответствовать описанию в задаче.
 **Исключения нужно только порождать, перехватывать их будет тестирующая система!**

## Примеры
### Пример 1
**Ввод:**
```
data = ['croaks', 'sweet-sweet-sweet',
        'sweeter-than-sweet',
        'trills', 'ee-oh-lay']
param = {'taboo': 'ee'}
print(chirp(*data, **param))
```
**Вывод:**
```
['croaks', 'trills']
```

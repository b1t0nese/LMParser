# Пятый элемент

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В файле `elements.txt` записаны элементы: вообще-то их должно быть четыре, но всякое бывает... Пятый элемент находится как сумма модулей разности первого и четвёртого и второго и третьего элементов.

Если вместо какого-либо числа записана строка, то пятым элементом считается число 0.

Если в файле не хватает элементов, то программа не падает с ошибкой, а выводит 1000 вместо пятого элемента.

В любом случае в конце выводится сообщение:
 `The fifth element has been found!`

Проверять условия нужно именно в описанном порядке (на случай, если встретятся оба варианта).

Использовать `if` нельзя, `try ... except` обязательно.

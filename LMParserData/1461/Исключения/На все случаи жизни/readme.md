# На все случаи жизни

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Напишите программу, которая может дать совет на все случаи жизни. Если же ответа нет, то она даёт универсальный совет, например, такой:
 `Делай, что должен, и будь, что будет.`

Использовать `if` нельзя, `try ... except` обязательно.

## Формат ввода
До пустой строки вводятся строки, в которых через символ табуляции записаны жизненная ситуация и совет.
 Затем снова до пустой строки вводятся ситуации.

## Формат вывода
Если введена ситуация, для которой не было совета, выводите универсальный совет.

## Примеры
### Пример 1
**Ввод:**
```
неизвестность	исследуй
слабость	будь сильным
грусть	читай книги
некомфортно	преодолевай трудности
плохие вещи	наполни жизнь хорошими
разбитая мечта	мечтай снова

неизвестность
грусть
глупость
обида
разбитая мечта
```
**Вывод:**
```
исследуй
читай книги
Делай, что должен, и будь, что будет.
Делай, что должен, и будь, что будет.
мечтай снова
```

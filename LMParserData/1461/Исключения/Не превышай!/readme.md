# Не превышай!

**Ограничение времени:** 2 секунды
**Ограничение памяти:** 64 Мб

Чтобы не получить штраф, не нужно нарушать скоростной режим: в населённых пунктах скорость должна быть не больше 60 км/ч, если не установлены дополнительные ограничивающие знаки, вне населённых — 90.
 Однако, если превышение скорости не больше некоторой величины, то штраф вы ещё не получите.

Напишите функцию **not_exceed()**, которая выписывает штрафы (или нет).

Функция принимает произвольное количество словарей с ключами (если нет ключа — *ограничение скорости*, то в населённом пункте ограничение 60, вне — 90, если нет ключа — *населённый пункт*, считается, что едем вне пунктов):
 `settlement` — населённый пункт (`True` или `False`);
 `limit` — ограничение скорости;
 `speed` — текущая скорость;
 `delta` — допустимое превышение;
 `fine` — штраф.

Функция должна вернуть список штрафов по одному для каждого словаря. Если штраф не назначается, то значение 0. Если в одном из словарей в населённом пункте дополнительное ограничение скорости окажется больше 60, нужно возбудить собственное исключение `SpeedLimitError` с сообщением: *Speed limit controversy.*
 Если значение штрафа в одном из словарей равно нулю, нужно возбудить собственное исключение `DataError` с сообщением: *Error in data.*

Порядок проверки исключений должен соответствовать описанию в задаче.
 **Исключения нужно только порождать, перехватывать их будет тестирующая система!**

## Примеры
### Пример 1
**Ввод:**
```
data = [{
    'settlement': True,
    'speed': 75,
    'delta': 10,
    'fine': 500
},
    {
        'limit': 110,
        'speed': 95,
        'delta': 20,
        'fine': 1500
    },
    {
        'settlement': False,
        'speed': 105,
        'delta': 10,
        'fine': 300
    }
]
print(not_exceed(*data))
```
**Вывод:**
```
[500, 0, 300]
```

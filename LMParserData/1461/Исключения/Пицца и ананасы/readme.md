# Пицца и ананасы

**Ограничение времени:** 2 секунды
**Ограничение памяти:** 64 Мб

Приготовим пиццу из имеющихся продуктов. Для этого напишем функцию **pizza()**, принимающую произвольное число строк-продуктов и именованные аргументы:
 `vegan` — веганская или нет; если веганская (`True`), то исключаются продукты, содержащие `meat`;
 `sauce` — с соусом или без; если аргумент передан, то указанный в нём соус добавляется к пицце сразу после теста.

Функция должна вернуть список продуктов для пиццы в такой последовательности:

- сначала кладётся тесто (`dough`). Если такого продукта не оказалось, возбуждается собственное исключение `PizzaWithoutDoughError` с сообщением: *You can't make pizza without dough.*
- затем добавляется соус, если передан соответствующий аргумент; если аргумент передан, а соуса среди продуктов нет, то возбуждается собственное исключение `SauceLackError`, которое наследуется от `PizzaWithoutDoughError`, с сообщением: *Where's the sauce?*
- теперь кладём сыр — если в продуктах сыров несколько (строка, содержащая cheese), то первый добавляем сейчас, последний — последним продуктом; если сыров меньше двух, возбуждается собственное исключение `NotEnoughCheeseError`, которое наследуется от `PizzaWithoutDoughError`, с сообщением: *You're not using enough cheese on that pizza.*
- проверяем, не затесался ли среди продуктов ананас; если есть строки, содержащие `ananas` или `pineapple`, то возбуждается собственное исключение `AnanasError`, которое наследуется от `PizzaWithoutDoughError`, с сообщением: *Pineapple on pizza!*
- если пицца должна быть веганская, то из продуктов убираем все, содержащие `meat`; если нет, то не убираем.

Теперь оставшиеся продукты сортируем сначала по убыванию длины строки, затем по алфавиту и кладём между сырами. Если класть нечего, возбуждается собственное исключение `OutOfGroceriesError` с сообщением: *There's nothing to put in the pizza.*

Приятного аппетита!

Порядок проверки исключений должен соответствовать описанию в задаче.
 **Исключения нужно только порождать, перехватывать их будет тестирующая система!**

## Примеры
### Пример 1
**Ввод:**
```
data = ['sausage meat', 'egg', 'cheese', 'dough',
        'champignons', 'tomato', 'garlic', 'ketchup',
        'gorgonzola cheese']
params = {'vegan': False, 'sauce': 'ketchup'}
print(pizza(*data, **params))
```
**Вывод:**
```
['dough', 'ketchup', 'cheese', 'sausage meat', 'champignons', 'garlic', 'tomato', 'egg', 'gorgonzola cheese']
```

# Не треугольник

**Ограничение времени:** 2 секунды
**Ограничение памяти:** 64 Мб

Напишите функцию **istriangle()**, которая проверяет, что три кортежа переданных ей координат точек образуют треугольник.

Если передано меньше трёх кортежей координат, нужно возбудить собственное исключение `NotEnoughError` с сообщением: *Not enough arguments.*

Если хотя бы в одном из кортежей не две координаты, возбудите исключение `DoNotMatchError` с сообщением: *The number of coordinates does not match.*

Функция возвращает булево значение: является ли фигура треугольником.

Приоритет проверки исключений должен соответствовать порядку описания в задаче.
 **Исключения нужно только порождать, перехватывать их будет тестирующая система!**

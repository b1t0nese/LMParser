# Расписание поезда

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

В файле `schedule.csv` записано расписание движения поезда с заголовками (разделитель — запятая):
`номер, станция, время в пути до станции часы, минуты`
`no, station, hour, minute`

Считая, что поезд едет со средней скоростью **60 км/ч**, рассчитайте расстояние, пройденное поездом до каждой станции от начала движения.

Выведите в отдельных строках названия станций и через символ табуляции расстояние в километрах, округлённое до целого вниз.

В примере ввода показано содержимое файла.

## Примеры
### Пример 1
**Ввод:**
```
no,station,hour,minute
1,Деревянка,13,20
2,Петрозаводск-Пасс,13,50
3,Кондопога,15,13
4,Кедрозеро,15,57
5,Кяппесельга,16,42
6,Медвежья Гора,17,22
7,Сегежа,20,3
```
**Вывод:**
```
Деревянка	800
Петрозаводск-Пасс	830
Кондопога	913
Кедрозеро	957
Кяппесельга	1002
Медвежья Гора	1042
Сегежа	1203
```

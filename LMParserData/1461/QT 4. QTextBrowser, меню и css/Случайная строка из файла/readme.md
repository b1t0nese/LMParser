# Случайная строка из файла

**Ограничение времени:** 1 секунда
**Ограничение памяти:** 64 Мб

Напишите интерфейс на PyQT — форма, наследник `QWidget` — для вывода случайной строки из файла `lines.txt` в `QTextBrowser`. Строку выводите заголовком первого уровня (тег h1).

Файл может быть пустым, тогда и выводить ничего не надо.

Класс, реализующий окно приложения, назовите `RandomString`. Кнопку — `button`, поле для вывода строки — `textBrowser`. Класс `RandomString` должен иметь определение метода `load_random_string`, который открывает файл и выводит содержимое в окно. В случае отсутсвия файла нужно выводить пустую строку.

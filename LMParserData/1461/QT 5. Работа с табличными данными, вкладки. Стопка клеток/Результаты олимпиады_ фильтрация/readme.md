# Результаты олимпиады: фильтрация

**Ограничение времени:** 10 секунд
**Ограничение памяти:** 64 Мб

В файле [rez.csv](https://yastatic.net/s3/lyceum/content/problems/qt6/rez.csv) записаны результаты олимпиады с заголовками (разделители — запятые):

```
место, имя участника, логин, названия четырех задач и сумма баллов
place,user_name,login,"1(Система счисления)","2(Количество символов)","3(Минимальное число)","4(Трамвай)","Score"
```

Логин участника имеет вид: sh-kaluga16-SS-CC-N, где SS — номер школы, а CC — номер класса, N — номер участника.

Например:

```
1-4,"У 09 11 Иванов И","sh-kaluga16-09-11-1","100(+)","100(+)","100(+1)","0(+3)","300"
1-4,"У 01 09 Петров П","sh-kaluga16-01-09-1","100(+2)","100(+1)","100(+)","","300"
```

Напишите приложение на PyQT6, в окне которого отображается содержимое файла. Класс, реализующий окно приложения и наследующийся от`QWidget`, назовите *OlympResult*.

Для отображения таблицы используйте `QTableWidget` с именем *tableWidget*. С помощью двух выпадающих списков `QComboBox` обеспечьте возможность фильтрации по следующим значениям:

- Номер школы (назовите выпадающий список *schools*)
- Номер класса (назовите выпадающий список *classes*)

Возможные школы и классы необходимо получать из файла и отображать их в выпадающем списке фильтра. Фильтры могут накладываться друг на друга. То есть, если выбран только номер школы — необходимо отображать все классы данной школы, если только номер класса — классы с данным именем в каждой из школ, если и номер школы, и номер класса — данные по выбранному классу в выбранной школе.

Для запуска установленных фильтров используйте кнопку с именем *resultButton*.
